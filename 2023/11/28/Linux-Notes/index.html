文件操作

vim 编辑器

more

  • 空白键 (space)：代表向下翻一页；
  • Enter ：代表向下翻『一行』；
  • /字串 ：代表在这个显示的内容当中，向下搜寻『字串』这个关键字；
  • :f ：立刻显示出档名以及目前显示的行数；
  • q ：代表立刻离开 more ，不再显示该文件内容。
  • b 或 [ctrl]-b ：代表往回翻页，不过这动作只对文件有用，对管线无用。

less

  • 空白键 ：向下翻动一页；
  • [pagedown]：向下翻动一页；
  • [pageup] ：向上翻动一页；
  • /字串 ：向下搜寻『字串』的功能；
  • ?字串 ：向上搜寻『字串』的功能；
  • n ：重复前一个搜寻 (与 / 或 ? 有关！)
  • N ：反向的重复前一个搜寻 (与 / 或 ? 有关！)
  • q ：离开 less 这个程序；
