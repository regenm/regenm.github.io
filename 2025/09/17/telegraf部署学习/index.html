telegrafйғЁзҪІеӯҰд№ 

Telegraf жҳҜдёҖж¬ҫз”ұејҖжәҗе…¬еҸё InfluxData ејҖеҸ‘зҡ„д»ЈзҗҶзЁӢеәҸпјҢе®ғжҳҜзҺ°д»Јзӣ‘жҺ§зі»з»ҹвҖңTICK StackвҖқжҲ–вҖңTelegraf StackвҖқзҡ„ж ёеҝғж•°жҚ®ж”¶йӣҶз»„д»¶гҖӮе®ғзҡ„дё»иҰҒеҠҹиғҪжҳҜд»Һеҗ„з§ҚжқҘжәҗпјҲеҰӮж“ҚдҪңзі»з»ҹгҖҒж•°жҚ®еә“гҖҒзЎ¬д»¶дј ж„ҹеҷЁгҖҒж¶ҲжҒҜйҳҹеҲ—зӯүпјүйҮҮйӣҶжҢҮж Үе’Ңж•°жҚ®пјҢз„¶еҗҺе°Ҷе…¶жұҮиҒҡе№¶еҶҷе…ҘеҲ°еӨҡз§Қзӣ®зҡ„ең°пјҲжңҖеёёз”Ёзҡ„жҳҜж—¶еәҸж•°жҚ®еә“ InfluxDBпјүгҖӮTelegraf д»Ҙе…¶жҸ’д»¶еҢ–зҡ„жһ¶жһ„гҖҒдҪҺиө„жәҗж¶ҲиҖ—е’ҢејәеӨ§зҡ„йӣҶжҲҗиғҪеҠӣиҖҢй—»еҗҚпјҢжҳҜ DevOps е’Ңзі»з»ҹз®ЎзҗҶе‘ҳиҝӣиЎҢжҢҮж Үзӣ‘жҺ§е’ҢжҖ§иғҪеҲҶжһҗзҡ„йҰ–йҖүе·Ҙе…·д№ӢдёҖгҖӮ

telegrafйғЁзҪІзӣ‘жҺ§жң¬жңәдҝЎжҒҜе№¶жҢҒд№…еҢ–еҲ°influxdb

еҮҶеӨҮ telegraf.conf

docker run вҖ“rm telegraf telegraf config > telegraf.conf

дҝ®ж”№ telegraf.conf

иө·е®№еҷЁ

жЈҖжҹҘinfluxdb

жҲ‘иҝҷйҮҢйғЁзҪІзҡ„influxdbеңЁеҗҢдёҖеҸ°дё»жңәпјҢж–№ејҸд№ҹжҳҜе®№еҷЁеҢ–йғЁзҪІ

telegrafйғЁзҪІзӣ‘жҺ§mqttдё»йўҳе№¶жҢҒд№…еҢ–еҲ°influxdb

дҝ®ж”№д№ӢеүҚзҡ„ telegraf.conf

йҮҢйқўжңүеҫҲеӨҡи®ҫзҪ®пјҲй»ҳи®ӨжҳҜиў«жіЁйҮҠзҡ„пјүпјҢеҸӘйңҖдҝ®ж”№д»ҘдёӢеҶ…е®№еҚіеҸҜгҖӮ

йҮҚеҗҜе®№еҷЁеҗҺиҝӣиЎҢжөӢиҜ•

зҷ»еҪ•д№ӢеүҚзҡ„mqttе®№еҷЁ

дҪҝз”Ёdocker exec -it mosquitto shиҝӣе…Ҙе®№еҷЁеҶ…е‘Ҫд»ӨиЎҢе·Ҙе…·гҖӮд№ӢеҗҺжү§иЎҢд»ҘдёӢе‘Ҫд»ӨгҖӮ

з„¶еҗҺжЈҖжҹҘtelegrafзҡ„ж—Ҙеҝ—д»ҘеҸҠinfluxdbж•°жҚ®еә“гҖӮ

иҝҷдәӣжҠҘй”ҷжҳҜжҲ‘д№ӢеүҚзҡ„иҫ“е…Ҙзҡ„jsonдёҚеҗҲжі•зҡ„еҺҹеӣ гҖӮеҗҺжқҘеҸ‘дәҶеҮ жқЎдёҚеҶҚеҮәзҺ°ж–°зҡ„дҝЎжҒҜиҜҙжҳҺжІЎд»Җд№Ҳй—®йўҳдәҶгҖӮ

жҲ‘иҝҷйҮҢйғЁзҪІзҡ„influxdbеңЁеҗҢдёҖеҸ°дё»жңәпјҢж–№ејҸд№ҹжҳҜе®№еҷЁеҢ–йғЁзҪІ

еҸҜд»ҘеҸ‘зҺ°еӨҡеҮәжқҘдәҶдёҖдёӘmqtt_consumer пјҢиҝӣеҺ»зңӢзңӢж•°жҚ®жғ…еҶөгҖӮж•°жҚ®жӯЈеёёеҶҷе…ҘгҖӮиҮіжӯӨз»“жқҹгҖӮ

еҸҜд»ҘеҹәдәҺжӯӨе®һзҺ°жӣҙеӨҡиҮӘе·ұзҡ„жғіжі•дәҶгҖӮ
