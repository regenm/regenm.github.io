1.3 grafana创建可视化模版

​ 在 Grafana 添加 Prometheus 数据源：

  1. 打开 Grafana → Configuration → Data Sources → 添加 Prometheus。
  2. URL 填写 http://<Prometheus服务器IP>:9090。
  3. 导入官方的 Node Exporter Dashboard（ID: 1860）
    • 在 Grafana → Dashboard → Import → 输入 1860 → 选择数据源。
