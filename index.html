汇编语言基本信息

​ 1.汇编语言（ assembly language）是二进制指令的文本形式，与指令是一一对应的关系。

​ 2.每一种 CPU 的机器指令都是不一样的，因此对应的汇编语言也不一样。例如: 最常见的INTEL X86汇编语言。

寄存器

寄存器种类

​ 1.早期的 X 86架构CPU只有8个寄存器，ESP 寄存器有特定用途，保存当前 Stack 的地址。现在都变成通用寄存器。

  • -  EAX
-  EBX
-  ECX
-  EDX
-  EDI
-  ESI
-  EBP
-  ESP
    1. 32位 CPU、64位 CPU 这样的名称，其实指的就是寄存器的大小。32 位 CPU 的寄存器大小就是4个字节。

内存模型：heap

​ 1.从低位到高位。

​ 2.用户主动请求而划分出来的内存区域，叫做 Heap（堆）。必须手动释放，或者由垃圾回收机制来回收。

​

内存模型：Stack

​ 1.除了 Heap 以外，其他的内存占用叫做 Stack（栈）。

​ 2.函数执行时，会为它在内存里面建立一个帧（frame）。一般来说，调用栈有多少层，就有多少帧。

​ 3.所有的帧都存放在 Stack，由于帧是一层层叠加的，所以 Stack 叫做栈。

​ 4.生成新的帧，叫做”入栈”，英文是 push；栈的回收叫做”出栈”，英文是 pop。

​ 5.Stack 的特点就是，最晚入栈的帧最早出栈（因为最内层的函数调用，最先结束运行），这就叫做”后进先出”的数据结构。每一次函数执行结束，就自动释放一个帧，所有函数执行结束，整个 Stack 就都释放了。
